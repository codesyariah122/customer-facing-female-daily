# Starter

A simple starter to create HTML with Vitejs (Hot-reload) + Tailwind 3

## Specs

Node >= 16

## How to Run

```bash
$ yarn install
OR
$ npm install

```

```bash
$ yarn dev
OR
$ npm run dev

```

then open http://localhost:3002 on browser
